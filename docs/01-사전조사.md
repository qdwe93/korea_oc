## 3. 도메인 / 호스팅 현황

- **도메인**: koreaoilchem.com — 정상 보유·운영 중 (DNS 응답: 211.234.116.139)
- **현재 사이트 상태**: 루트가 `temp.php`로 리다이렉트되는 **"Under Construction" 임시 페이지**.
  푸터에 `Copyright © 2011 ... Designed by GAUTECH` 표기 → **2011년부터 약 15년간 임시 페이지 상태로 방치**.
- **호스팅**: HTTPS 접속 시 `gautech.net` 인증서가 응답 → 제작사(GAUTECH)의 공유 호스팅 서버에 얹혀 있는 것으로 추정.
  - PHP가 구동되는 전형적인 국내 웹호스팅 환경 → **정적 HTML 업로드만으로 새 사이트 배포 가능**.
  - **자체 도메인용 SSL 인증서 없음** → 새 사이트 오픈 시 HTTPS 적용 필요 (판단 근거는 기획서 참조).
- **메일 서버**: mail.koreaoilchem.com 운영 흔적 → 도메인 메일은 유지해야 하므로, 호스팅/DNS 변경 시 **MX 레코드 보존 필수**.

## 5. 업종·입지에서 추론한 사업 성격 (추정 — 확인 필요)

- **업종 코드가 "액체연료 및 관련제품 도매업"** → 최종 소비자 대상이 아닌 **B2B 유통·무역업**.
- 소재지가 **부산 동구 초량동(부산항 북항 바로 앞, 무역·해운 사무실 밀집 지역)**, 영문 사명에 "OIL CHEMICAL" 사용, 자체 도메인 메일 운영 → **선박용 연료(벙커유)·윤활유·석유화학제품 등의 유통 또는 수출입 중개**일 가능성이 높다.
- 다만 **구체적 취급 품목·거래처·연혁은 공개된 자료가 전혀 없음** → 대표 인터뷰(또는 의뢰인 확인)로 채워야 하며, 그 전까지 PRD에는 플레이스홀더로 처리한다.

## 6. 웹 검색 결과 요약

- "코리아유화", "koreaoilchem", "서용환 + 유화" 등 어떤 조합으로도 **회사 관련 공개 콘텐츠(뉴스, 채용, SNS, 디렉토리)가 검색되지 않음**. 유사명 기업(대한유화, 한국유화 등)만 검색됨.
- 시사점 ①: **인터넷에서 구할 수 있는 회사 실사진·자료는 사실상 없음** → 이미지는 전량 생성 또는 직접 촬영으로 준비해야 함 (→ `04-이미지-준비목록.md`).
- 시사점 ②: 검색 노출이 0에 가까우므로, 새 홈페이지는 **"코리아유화" 검색 시 첫 결과로 잡히는 것** 자체가 큰 성과가 된다 (SEO 기본기만으로 달성 가능).
