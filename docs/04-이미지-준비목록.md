# 04. 이미지 준비 목록 — 코리아유화 홈페이지

> 작성일: 2026-07-07
> 전제: 사전조사 결과, 인터넷에서 구할 수 있는 회사 실사진은 **없다**. 따라서 로고 원본(저해상도)을 제외한 모든 이미지는 **AI 생성**으로 준비한다. 아래 프롬프트로 생성 후 `assets/images/`에 지정된 파일명으로 저장하면 개발 단계에서 그대로 연결된다.

## 공통 스타일 가이드 (모든 프롬프트에 적용됨)

- 톤: 사진 같은(photorealistic) 기업 이미지. 네이비/블루 위주, 새벽·주간의 깨끗한 광. 오렌지빛 포인트는 소량.
- **이미지 안에 글자·로고·워터마크가 생기지 않도록** 프롬프트에 명시했다 (생성 AI의 텍스트는 대부분 깨짐).
- 사람 얼굴 클로즈업 회피 (초상권 오해 + 어색함 방지).
- 생성 비율을 지원하지 않는 도구면 큰 쪽으로 생성 후 크롭.

## A. 생성이 필요한 이미지 (우선순위 순)

### A-1. 홈 히어로 `hero-main.jpg` — 1920×1080 (16:9) ★필수
용도: 홈 첫 화면 전체 배경. 위에 흰색 헤드라인이 올라가므로 상단이 어둡고 차분해야 함.

```
A wide cinematic photograph of a large oil tanker ship docked at Busan
harbor at dawn, deep navy blue sea and sky, industrial port cranes in
silhouette, soft golden-orange sunrise light on the horizon, calm
professional corporate atmosphere, photorealistic, high detail,
muted colors, darker toward the top of the frame for text overlay,
no text, no watermark, no logos
```

### A-2. 사업영역: 액체연료 유통 `biz-fuel.jpg` — 1200×800 (3:2) ★필수
용도: 사업영역 페이지 카드 ①.

```
A photorealistic image of industrial fuel storage tanks and a tanker
truck at a clean modern oil depot, blue sky, steel silver tanks with
navy blue accents, pipelines in foreground, bright daylight, corporate
photography style, no people, no text, no watermark, no logos
```

### A-3. 사업영역: 선박·항만 공급 `biz-marine.jpg` — 1200×800 (3:2) ★필수
용도: 사업영역 페이지 카드 ②. (취급 품목 확정 전이지만 부산항 입지 기반 추정 — 품목 확정 후 교체 가능)

```
A photorealistic image of a bunkering barge refueling a large cargo
ship at a busy container port, deep blue ocean, container cranes in
the background, clear daytime, professional maritime industry
photography, no text, no watermark, no logos
```

### A-4. 사업영역: 화학제품·윤활유 `biz-chem.jpg` — 1200×800 (3:2) ★필수
용도: 사업영역 페이지 카드 ③.

```
A photorealistic image of neatly stacked industrial chemical drums and
lubricant containers in a bright clean warehouse, navy blue and silver
color scheme, orderly rows, soft industrial lighting, corporate
photography style, no people, no text, no watermark, no logos
```

### A-5. 회사소개 상단 `about-visual.jpg` — 1600×600 (와이드 배너) ★필수
용도: 회사소개 페이지 상단 배너. 텍스트가 올라가므로 단순한 구도.

```
A wide panoramic photograph of Busan port skyline at dusk seen from
across the water, navy blue tones, ship lights reflecting on calm sea,
minimal composition with large open sky area, serene and trustworthy
corporate mood, photorealistic, no text, no watermark
```

### A-6. 대표 인사말 배경 `about-greeting.jpg` — 800×1000 (세로) (선택)
용도: 인사말 옆 세로 이미지. 대표 실제 사진이 있으면 그것으로 교체 (실사진이 항상 우선).

```
A photorealistic vertical image of a calm office desk by a large window
overlooking a harbor, fountain pen on documents, soft morning light,
navy and warm gray tones, professional and sincere atmosphere,
no people, no readable text on documents, no watermark
```

### A-7. OG(공유 미리보기) 이미지 `og-image.jpg` — 1200×630 ★필수
용도: 카톡/메신저에 URL 공유 시 미리보기. **A-1 히어로를 1200×630으로 크롭해 재사용** — 별도 생성 불필요. (로고 텍스트 합성은 개발 단계에서 CSS/캔버스로 처리)

## B. 생성이 아닌 방법으로 준비하는 것

| 항목 | 방법 | 이유 |
|---|---|---|
| 로고 | `assets/reference/temp_p01.jpg`의 기존 로고를 근거로 **텍스트+SVG로 리메이크** (개발 단계에서 코드로 제작) | 생성 AI는 정확한 글자를 못 그림. 기존 문안("KOREA OIL CHEMICAL CO.,LTD" + 국문)과 배색(네이비→스카이블루)을 그대로 계승해야 하므로 코드 제작이 정확함 |
| 파비콘 | 리메이크 로고에서 "K" 모노그램 추출 (SVG) | 동일 |
| 오시는길 지도 | 카카오맵 퍼가기(iframe) | 실제 지도가 필요, 이미지 아님 |
| 대표 사진(선택) | 실물 촬영 제공 시에만 게재 | 인물 생성 이미지는 신뢰 목적에 정면으로 반함 — **대표 얼굴을 AI로 만들지 않는다** |
| 사무실/현장 사진(선택) | 보유 시 제공 | 실사진이 생기면 A-2~A-4를 교체하는 것을 권장 |

## C. 파일 배치 규칙

```
assets/
  reference/    ← 기존 사이트에서 수집한 원본 (완료)
  images/       ← 생성 이미지를 위 파일명 그대로 저장
    hero-main.jpg
    biz-fuel.jpg
    biz-marine.jpg
    biz-chem.jpg
    about-visual.jpg
    about-greeting.jpg   (선택)
```

- 개발 시 `<img>` 경로를 위 파일명으로 미리 잡아두므로, **이미지를 폴더에 넣기만 하면 사이트에 반영**된다.
- 이미지가 준비되기 전에는 브랜드 컬러의 CSS 그라데이션 플레이스홀더로 표시되도록 구현한다 (이미지 없어도 사이트가 깨지지 않게).
- 용량 가이드: 웹 업로드 전 JPG 품질 80 내외, 히어로 400KB 이하 권장 (개발 단계에서 일괄 최적화 예정이므로 원본은 크게 생성해도 무방).
